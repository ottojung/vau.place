  • Have you seen Kernel?

    I like Kernel. It is such a simple language, yet so beautifully powerful. It is a lisp, you know, but more in the schemey-way. So there is homoiconicity and macros and continuations, all backed into the language. But, unlike other lisps, Kernel has it all first-class!
